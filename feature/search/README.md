# Модуль :feature:search

Экран "Поиск" — поиск треков, артистов, альбомов и плейлистов.

## Назначение

- Поисковая строка с автодополнением
- Результаты поиска по категориям
- История поиска (TODO)

## Зависимости

```
:core:ui   — тема и компоненты
:core:data — репозиторий данных
```

## Ключевые файлы

| Файл | Описание |
|------|----------|
| `SearchScreen.kt` | Экран поиска |
| `SearchViewModel.kt` | ViewModel (TODO) |

## API эндпоинты

- `GET /search?text={query}&type={type}` — поиск
- `GET /search/suggest?part={query}` — автодополнение

**Типы поиска:**
- `all` — всё (по умолчанию)
- `track` — треки
- `artist` — артисты
- `album` — альбомы
- `playlist` — плейлисты

## Задачи для ИИ-агента

При работе с этим модулем:
- Debounce для поисковых запросов (300-500ms)
- Показывать suggestions при вводе
- Кэшировать историю поиска локально
