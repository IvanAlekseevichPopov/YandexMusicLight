# Модуль :feature:home

Экран "Главная" — лента рекомендаций и персональные плейлисты.

## Назначение

- Отображение фида пользователя
- Персональные плейлисты (Плейлист дня, Премьера и т.д.)
- Чарт
- Новые релизы

## Зависимости

```
:core:ui   — тема и компоненты
:core:data — репозиторий данных
```

## Ключевые файлы

| Файл | Описание |
|------|----------|
| `HomeScreen.kt` | Главный экран |
| `HomeViewModel.kt` | ViewModel (TODO) |

## API эндпоинты

- `GET /feed` — фид рекомендаций
- `GET /landing3/chart` — чарт
- `GET /landing3/new-releases` — новые релизы

## Задачи для ИИ-агента

При работе с этим модулем:
- Экран использует `MusicRepository` из `:core:data`
- Состояние управляется через ViewModel
- UI компоненты импортировать из `:core:ui`

**Планируемая структура экрана:**
```
┌─────────────────────────┐
│  Плейлист дня          │ ← Горизонтальный список
│  [🎵] [🎵] [🎵] →      │
├─────────────────────────┤
│  Чарт                   │
│  1. Track name          │
│  2. Track name          │
│  3. Track name          │
├─────────────────────────┤
│  Новые релизы           │
│  [📀] [📀] [📀] →      │
└─────────────────────────┘
```
