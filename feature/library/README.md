# Модуль :feature:library

Экран "Моя музыка" — библиотека пользователя.

## Назначение

- Лайкнутые треки
- Сохранённые альбомы
- Подписки на артистов
- Плейлисты пользователя

## Зависимости

```
:core:ui   — тема и компоненты
:core:data — репозиторий данных
```

## Ключевые файлы

| Файл | Описание |
|------|----------|
| `LibraryScreen.kt` | Экран библиотеки |
| `LibraryViewModel.kt` | ViewModel (TODO) |

## API эндпоинты

- `GET /users/{uid}/likes/tracks` — лайкнутые треки
- `GET /users/{uid}/likes/albums` — сохранённые альбомы
- `GET /users/{uid}/likes/artists` — подписки на артистов
- `GET /users/{uid}/playlists/list` — плейлисты пользователя

## Задачи для ИИ-агента

При работе с этим модулем:
- `uid` получать из `/account/status`
- Поддержка пагинации для длинных списков
- Кэширование для оффлайн доступа
