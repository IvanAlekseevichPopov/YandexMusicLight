# Модуль :app

Главный модуль приложения. Точка входа и связывание всех feature-модулей.

## Назначение

- `MainActivity` — точка входа, инициализация Compose
- `Navigation` — настройка навигации между экранами
- Связывание feature-модулей через Navigation Compose

## Зависимости

```
:core:ui        — тема и общие компоненты
:core:data      — репозитории данных
:core:network   — сетевой слой
:feature:home   — экран главной
:feature:search — экран поиска
:feature:library — экран библиотеки
```

## Ключевые файлы

| Файл | Описание |
|------|----------|
| `MainActivity.kt` | Точка входа, Scaffold с нижней навигацией |
| `navigation/Navigation.kt` | Определение маршрутов и элементов навигации |

## Задачи для ИИ-агента

При работе с этим модулем:
- Не добавлять бизнес-логику — она должна быть в feature-модулях
- Navigation routes определяются здесь
- При добавлении нового экрана: добавить route в `Navigation.kt` и зависимость на feature-модуль
